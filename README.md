# italk.gr — Γλωσσάρι Διερμηνέα Ασύλου

Πολύγλωσσο γλωσσάρι ορολογίας ασύλου και πρώτης υποδοχής για διερμηνείς
στην Ελλάδα. Δωρεάν, χωρίς λογαριασμό, με αναφορά σε δημόσιες πηγές

- **Γλώσσες:** Ελληνικά · English · العربية · فارسی

## English

Multilingual asylum-interpreter glossary for Greek First Reception Centres,
EUAA work, and humanitarian contexts. Static HTML/CSS/JS — no backend,
no tracking, no account required. Built by a working interpreter, for
interpreters.

## Structure

- `index.html` — main glossary page
- `sources.html` — public sources used for terminology
- `about.html` — about the project
- `data/terms.json` — all term entries
- `app.js`, `styles.css` — frontend
- `CNAME` — GitHub Pages custom-domain marker
